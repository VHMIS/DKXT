# Website tuyển sinh 2015
Website tuyển sinh 2015 của trường CĐ Công nghệ thông tin Hữu nghị Việt Hàn 
## Build
###Chỉnh sửa file config

     destination:    /var/html/www/tuyensinh/ // web folder
     host:           0.0.0.0 // host
     port:           9001 // port
     baseurl:        /dk // web path
     url:            http://localhost // web link
###Build

     jekyll build
